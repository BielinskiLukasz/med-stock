  <behavior>
    Function signature: calculateStatus(med: Medicine, now: Date = new Date()): MedicineStatus

    Decision tree (in order):
    1. D-13: if med.manualStatus is non-null → return med.manualStatus (manual override wins)
    2. Derive: expiry = med.expiryDate ? new Date(med.expiryDate) : null
    3. Derive: opened = med.openedDate ? new Date(med.openedDate) : null
    4. Derive: paoEnd = opened && med.pao ? addPAO(opened, med.pao) : null
    5. D-14: if !expiry && paoEnd → return 'Opened' if now <= paoEnd, else 'Expired'
    6. D-15: if expiry && opened && !med.pao → return 'Expired' if now > expiry, else 'Opened'
    7. Standard: if expiry && now > expiry → 'Expired'; if paoEnd && now > paoEnd → 'Expired'; if opened → 'Opened'; default → 'Active'

    addPAO helper: adds pao.value in pao.unit (days: setDate, weeks: setDate + *7, months: setMonth) to a Date copy.

    Test cases (all must pass before implementation is complete):
    - Active: expiryDate '2030-12-31', not opened, now = 2026-06-30 → 'Active'
    - Opened: expiryDate '2030-12-31', openedDate '2026-06-01', pao {value:30,unit:'days'}, now = 2026-06-15 → 'Opened'
    - Expired by expiry: expiryDate '2020-01-01', now = 2026-06-30 → 'Expired'
    - Expired by PAO: expiryDate '2030-12-31', openedDate '2026-06-01', pao {value:30,unit:'days'}, now = 2026-07-15 → 'Expired'
    - Whichever first: expiryDate '2026-07-01' (sooner), openedDate '2026-06-01', pao {value:90,unit:'days'} (paoEnd 2026-09-01), now = 2026-07-15 → 'Expired' (expiry fires first)
    - D-13 manual override: expiryDate '2030-12-31', manualStatus 'Archived', now = 2026-06-30 → 'Archived' (not 'Active')
    - D-13 manual override with expired dates: expiryDate '2020-01-01', manualStatus 'Used Up', now = 2026-06-30 → 'Used Up' (not 'Expired')
    - D-14 null expiry + PAO within window: expiryDate null, openedDate '2026-06-01', pao {value:30,unit:'days'}, now = 2026-06-15 → 'Opened'
    - D-14 null expiry + PAO past window: expiryDate null, openedDate '2026-06-01', pao {value:30,unit:'days'}, now = 2026-07-15 → 'Expired'
    - D-15 opened + no PAO, expiry future: expiryDate '2026-12-31', openedDate '2026-06-01', pao null, now = 2026-06-30 → 'Opened'
    - D-15 opened + no PAO, expiry past: expiryDate '2020-01-01', openedDate '2019-06-01', pao null, now = 2026-06-30 → 'Expired'
